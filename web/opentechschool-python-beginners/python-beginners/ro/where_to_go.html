Unde să mergi în continuare¶

Să învățăm Python¶

Sperăm că acest curs te-a învățat destul Python pentru a te descurca pe cont propriu. Exista însă mult mai multe lucruri pe care le poți învăța! Chiar și programatorii experimentați vor încerca întotdeauna să învețe mai multe despre limbajul lor și despre cum să scrie un cod bun.

Cărți și Cursuri¶

Dacă îți place să înveți din cărți, există o gramadă de cărți bune, și chiar multe care poti fi citite gratuit pe Internet!

  • OTS Python Portal - Verifică și restul cursurilor și viitoare ateliere!

  • O’Reilly publică sute de cărți tipărite sau în format electronic, despre Python și multe alte tehnologii (verifică Learning Python, 5th edition).

  • Invent with Python are o abordare practică, cu trei cărți electronice diferite care pot fi citite gratuit.

  • Learn Python the Hard Way este un curs complet pas cu pas în limbaj, făcut într-un stil unic.

  • Dive into Python 3 este o altă carte disponibilă gratuit!

  • Ultimul dar nu cel din urmă, python.org are Its own tutorial!

Cursuri online¶

Este incredibil căt de multe poți face pe Internet în zilele noastre. Există multe cursuri de programare pe aici!

  • Edx - cu universități de top ca Stanford, MIT, Harvard și Berkley care pun la dispoziție gratuit cursuri, este greu să refuzi. Trebuie să vezi ce va mai apărea!

  • Coursera este asemănător cu Edx, cu și mai multe cursuri, și o pate din ele pot fi urmărite oricând.

  • CodeAcademy - din fericire acesta are un curs specific Python care poate fi urmat oricând, și multe cursuri practice pentru alte limbaje.

Ce poți face cu Python¶

Asta chiar e o întrebare dificilă! Vezi tu, practic orice îți vine în minte care implică ceva electronic - de la telecomanda ta pentru televizor, până la un telefon inteligent, până la ce este in spatele unui serviciu web, până la programarea aterizărilor pe un aeroport, până la programele pe care le vezi în fiecare zi - toate acestea implică programare pe undeva!

Și nu se limitează doar la latura practică, dar și la cea tehnică și academică. Imediat ce ai înțeles bazele unui limbaj de uz general ca Python poți începe să lucrezi la diverse lucruri: interfețe web, servicii de backend web, analiză de date și statistică, Inteligență Artificială, design GUI (Graphical User Interface - Interfață Grafică Utilizator), robotică, dezvoltare software de toate felurile, tranzacții online, automatizări și multe altele.

Nu ți-o luăm în nume de rău dacă îți sună descurajator, așa că iată niște sfaturi practice pentru a înțelege cât de larg este domeniul “programare”:

Aplică noile tale competențe pentru orice te pasionează.

Fie că este vorba de a începe o afacere, ajutându-te să execuți niște prelucrări de date mai rapid, să pui un site web online, să calculezi șansele într-un joc pe care îl joci, să creezi artă și muzică, sau orice altceva, să descoperi cum poți folosi programarea pentru a te ajuta la ce ai tu nevoie o să transforme învățarea în ceva mult mai atractiv și relevant. Nu ezita să întrebi pe altcineva cu mai multă experiență dacă nu știi cum să începi, dar întotdeauna gândește-te la ce vrei tu să obții!
