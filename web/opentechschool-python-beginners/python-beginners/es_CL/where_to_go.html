Donde ir desde aqui¶

Aprendiendo Python¶

Esperamos que este tutorial le haya enseñado lo sufieciente de python para ponerlo en buen camino. Sin embargo, hay mucho más que puede aprender! Incluso programadores profesionales siempre estarán tratando de aprender más acerca del lenguaje y como escribir código de manera exelente.

Libros y Tutoriales¶

Si le gusta aprender desde un libro, hay muy buenos libros, incluso muchos de ellos se pueden leer gratis en la web!

  • OTS Python Portal- Revise el resto de nuestros tutoriales, y espere por futuros talleres!

  • O’Reilly Publica cientos de libro y ebooks, sobre python y muchas otras tecnologías (vea Learning Python, 5th edition).

  • Invent with Python toma un enfoque práctico, con tres libros diferentes que pueden leerse gratis en la web.

  • Learn Python the Hard Way es un completo tutorial paso a paso sobre el lenguaje, escrito en un estilo único.

  • Dive into Python 3 es otro gran libro disponible gratis en línea!

  • Por último, pero no menos importante, python.org tiene su propio tutorial!

Cursos de aprendizaje en línea¶

No creerá lo que puede hacer en las páginas web estos días. Hay muchos cursos de programación en toda la red!

  • Edx- con las mejores universidades como Stanford, MIT, Harvard y Berkley dando cursos interactivos gratis, es difícil no tentarse con uno de ellos. Debe buscar los que empezarán pronto!

  • Coursera es muy parecido a Edx, con incluso más cursos, y algunos de ellos se pueden tomar en cualquier fecha.

  • CodeAcademy-afortunadamente este sitio tiene un curso específico sobre Python que puede ser tomado cuando quiera, y muchos otros cursos prácticos de otros lenguajes.

Que hacer con Python¶

Bueno, es una difícil pregunta! Practicamente cualquier cosa que involucre algo de electrónica- desde el control remoto de la tv, a un smartphone, al soporte de un popular servicio web, calendarización de aterrizajes de aviones en aeropuertos, o el software que utilice día a día - todo involucra algo de programación en alguna parte!

Y no es solamente un gran campo en la práctica, sino también técnica y académicamente. Tan pronto como tenga una comprensión básica de un lenguaje general como python, puede empezar a trabajar en todo tipo de ámbitos: portales web, soporte bajo un servicio web, análisis de datos y estadísticas, inteligencia artificial, diseño de GUI, robótica, desarrollo de software de todo tipo, traductores en línea, automatización de todo y mucho más.

No le culpamos si todo esto suena un poco desalentador, pero le daremos un consejo práctico, que toma en cuenta la ventaja de lo aplio que es el campo de la “programación”:

Aplique sus nuevas habilidades a todo lo que le apasione.

Ya sea que va a iniciar un nuevo negocio, ayude a realizar tareas que involucren analizar datos más rapido, ponga en línea un sitio web, calcule las ventajas en un juego que este jugando, cree arte o música, o cualquier cosa; averiguar cómo usar la programación para ayudar con cualquier cosa que haga flotar su bote hará el aprendizaje mucho más interesante y relevante. Siéntase libre de consultar con alguien que tenga más experiencia si no sabe por donde empezar, pero siempre mencione que es lo que le emociona!
