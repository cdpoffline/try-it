Um so richtig mit der Hardware-Programmierung loszulegen, brauchst Du noch ein Hardware-Set.

Auf dieser Seite haben wir verschiedene Kits für Euch aufgelistet. Es gibt natürlich noch viel mehr Kits auf dem Markt und vergleichen kann sich lohnen. Denn auch wir mussten die Erfahrung machen, dass einige der günstigen Sets nicht funktionierten.

Anfängerset UNO 1

Ein ziemlich umfangreiches Set zu einem super Preis. Zwar handelt es sich nicht um ein original Arduino, ist aber kompatibel zum Arduino UNO R3.

Das Anfängerset UNO 1 gibt es bei Amazon für ca. 50 Euro.

Das Arduino Starter Kit

Das Arduino-Starter Kit ist das offizielle Starter Kit von Arduino. Es beinhaltet unter anderem ein Text-Display, einen Servo und einen Elektromotor. Es beinhaltet ein Projektbuch in englischer Sprache.

Zurzeit kostet das Arduino Starter Kit bei Amazon für ca. 89 Euro.
